# Security Policy

## Supported Versions

Obsidian is currently always used at the latest desktop Version. 
which is at the time writing windows build obsidian at ver. 1.6.7.
I do also sometimes use the Android Obsidian app also build at ver. 1.6.7.
currently being supported with security updates.


## Reporting a Vulnerability

To report a vurneability please create an issue on [github/philipprochazka/Obsidian-Workspaces/](https://github.com/philipprochazka/Obsidian-Workspaces/issues)


## known Issues and potential Vulnerability

Android "obsidian-Git extension" ssh is not supported at the obsidian android app.
Windows "Obsidian-Git extension" also does have issues with Reaching Git with ssh and seems to be not supported at the obsidian desktop app either.
Other versions have not been tested
